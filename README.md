# Arena-Builder
Мод на Terraria, который добавляет 3 предмета: 1 строит арену 600х200 блоков, 2 строит арену высотой в 100 блоков от левого края мира до игрока, 3 строит снежный биом.
Чтобы установить мод необходимо воспользоваться браузером модов в игре.
Чтобы получить предметы необходимо их создать: Для 1 - 10 деревянных платформ, для 2 - 100 деревянных платформ, для 3 - 100 снежных блоков. Рабочее место для создания всех предметов - верстак.
Во время использования предмета игра может (скорее всего всегда) зависнуть. Длительность может быть от 1 секунды до нескольких минут в зависимости от тяжести изменений в мире и технических характеристик устройства.
